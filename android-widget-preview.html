What changed

Hero progress ring. Drawn at runtime via Canvas.drawArc in DtwgWidgetProvider and pushed through RemoteViews.setImageViewBitmap — anti-aliased, rounded end-caps, and a dynamic centre readout (% / ✓).
Bold display numeral. 3/7 in 34sp sans-serif-black with negative tracking, paired with a 13sp "Next: …" muted label.
Status pill. Rounded chip that adapts copy (Ready · In progress · Complete) and surfaces the day's phase at a glance.
CTA pill. Vivid accent button with an internal highlight gradient. Tap target is the entire widget.
Material You. Android 12+ pulls all colors from system_accent1_* and system_accent3_* tokens, so the widget rehues with the wallpaper. Pre-12 devices fall back to a hand-tuned purple gradient.
System chrome alignment. Outer radius tracks @android:dimen/system_app_widget_background_radius on API 31+ so the DTWG widget visually fuses with the launcher's widget container.
Larger target size. 4×2 cells (≈288×132 dp) on Android 12+ — picker preview is the actual layout, not a generic screenshot.
